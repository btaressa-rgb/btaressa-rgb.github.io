A structured schedule of the World Wide Web project milestones and development phases from its inception at CERN through its public release.

Project Schedule

WWW Project Development Milestones
Period Milestone Status
March 1989 Original proposal submitted by Tim Berners-Lee to CERN management Complete
May 1990 Revised proposal co-authored with Robert Cailliau Complete
October 1990 Project formally approved; work begins on browser and server Complete
December 1990 First web server (httpd) and browser (WorldWideWeb) operational on NeXT Complete
January 1991 Line-mode browser released for non-NeXT platforms Complete
August 1991 Web made publicly available on the internet; announcement on alt.hypertext Complete
December 1991 First web server outside Europe goes live at SLAC Complete
January 1992 Line-mode browser version 1.1 released Complete
November 1992 26 web servers known to be running worldwide Complete
April 1993 CERN announces W3 technology will be free and in the public domain Complete
October 1993 Over 500 web servers in operation worldwide Complete
1994 World Wide Web Consortium (W3C) founded at MIT Complete
