Hypertext

Home > Hypertext

Hypertext is text which contains links to other texts. The term was coined by Ted Nelson around 1965. HyperMedia is a term used for hypertext which is not constrained to be text: it can include graphics, video and sound, for example. Apparently Ted Nelson was the first to use this term too.

Overview

Hypertext and HyperMedia are concepts, not products. A hypertext system allows authors to link information together, create paths through a corpus of related material, annotate existing texts, and create notes that point readers to either bibliographic data or the body of the referenced text.

The WorldWideWeb (W3) is a hypertext system. The W3 project was started at CERN in 1989 with the aim of giving universal access to a large universe of documents. The W3 project uses hypertext as the basis for its information system.

Hypertext Terms

A number of terms are used in the hypertext community. See the glossary of hypertext terms for definitions of the most commonly used terms, including the definition of hypertext itself.

History and Timeline

The concept of hypertext has a rich history stretching back to Vannevar Bush's 1945 article "As We May Think," which proposed the MEMEX device. See the hypertext timeline for a detailed chronology of key developments in hypertext and the World Wide Web.

Project Xanadu

Ted Nelson originally invented the word "hypertext" to describe a system of non-sequential writing. This long-standing interest evolved into Project Xanadu. The Xanadu project introduced several revolutionary concepts including unique addressing, transclusion, and royalty attribution that predate modern web protocols.

HyperMedia

See also: HyperMedia — an extension of hypertext that incorporates other media types such as graphics, audio, and video into the linked information space.
