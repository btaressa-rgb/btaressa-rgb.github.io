Quick Guide — Line Mode Browser

Home > Line Mode > User Guide > Quick Guide

A quick reference card for the most essential commands in the WWW Line Mode Browser. Keep this handy while navigating the World Wide Web from the command line.

Navigation Commands

Essential Navigation Commands
Command Shortcut Action
<number> RETURN — Follow link number
back b Go back one document
home — Return to home document
top — Go to top of document
next RETURN Scroll down one page
up — Scroll up one page

Search Commands

Search and Find Commands
Command Example Description
find <keywords> find sgml html Search index for keywords
f <keywords> f hypertext Short form of find
go <address> go http://info.cern.ch Navigate to a URI
list — List all links in document

Utility Commands

Utility and Help Commands
Command Shortcut Action
help h Display help information
verbose v Toggle verbose mode
source — View HTML source of document
print — Print the current document
quit q Exit the browser
