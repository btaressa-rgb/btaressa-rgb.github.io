Line Mode Browser

Home > Executive Summary > Line Mode Browser

The WWW Line Mode Browser was the first cross-platform web browser, developed at CERN in 1991 by Nicola Pellow. Unlike the original WorldWideWeb browser which required a NeXT workstation, the line-mode browser ran on any computer with a simple terminal interface.

Overview

The line-mode browser was a crucial step in making the World Wide Web accessible to the wider internet community. It ran on Unix, VMS, and other operating systems, allowing researchers worldwide to access the Web without specialized hardware.

The browser displays documents as plain text with numbered references in [brackets]. Users navigate by typing the reference number and pressing RETURN.

Key Features

  • Cross-platform: runs on Unix, VMS, and other terminal-based systems
  • Supports HTTP, FTP, Gopher, and NNTP protocols
  • Numbered hypertext links for keyboard navigation
  • Keyword search for index documents
  • Command-line interface suitable for scripting and automation
  • No graphical display required

Using the Browser

For complete instructions on using the line-mode browser, see the User Guide. For a quick reference, see the documentation ideas page.

You can also try the browser yourself using the Line Mode Browser Simulator — a web-based recreation of the original experience.

Historical Significance

The line-mode browser's release in 1991 was a pivotal moment for the Web. By making the Web accessible on any internet-connected computer, it enabled the rapid growth of the early web community. The browser was instrumental in spreading awareness of the World Wide Web beyond CERN and the NeXT user community.
