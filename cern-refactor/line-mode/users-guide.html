WWW Line Mode Browser

Home > Line Mode > User Guide

The World-Wide Web line-mode browser allows you to find information by following references and/or by using keywords. It is the first cross-platform browser for the World Wide Web, running on any system with a terminal interface.

Following References

References are numbers in [brackets] after particular phrases. Type the number and RETURN for more information on the phrase.

For a quick guide to the most common commands, see the Quick Reference card.

Searching Index Documents

Some documents are indexes. These contain little text, but allow you to search for information with keywords. Type find or f (space) and the keywords.

For example:

f sgml examples

This searches the index for items with keywords SGML and EXAMPLE. You can only use the find command when it is present in the prompt. You can omit the f if the first keyword doesn't conflict with existing commands.

Common Commands

Line Mode Browser Commands
Command Description
<number> Follow the link with that reference number
back or b Go back to the previous document
home Go to the home document
top Go to the top of the current document
up Scroll up one page
next Scroll down one page (or press RETURN)
find <keywords> Search an index document for keywords
go <address> Navigate to a specific URI address
list List all links in the current document
help Display help information
quit or q Exit the browser
