Early Server Landscape

This list identifies early WorldWideWeb servers. Note that a single server machine often served multiple databases across different protocols.

Institution Location & Details Status
NCSA National Center for Supercomputing Applications, Urbana-Champaign, IL, USA. EXPERIMENTAL
Cornell Law Legal Information Institute: US Intellectual Property Statutes online. Active
SLAC Stanford Linear Accelerator Center, California. High-Energy Physics preprints. Active
DESY Deutsches Elektronen-Synchrotron, Hamburg, Germany. High-Energy Physics database. Active
CWI Center for Mathematics and Computer Science, Amsterdam. FTP hypertext. Active
NIKHEF Dutch High-Energy Physics center. Active
Fermilab Documentation for online and offline groups (FNAL). EXPERIMENTAL
